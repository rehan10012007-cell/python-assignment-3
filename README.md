# Library Inventory Manager

This is a command-line based Library Inventory Manager built in Python. It helps to manage books in a campus library using object-oriented programming and JSON file storage.

## Features

- Add new books with title, author and ISBN
- Issue and return books
- View all books
- Search books by title or ISBN
- Persistent storage using a JSON file
- Logging of operations

## Project Structure

- `library_manager/book.py` – Book class
- `library_manager/inventory.py` – Inventory manager class
- `cli/main.py` – Menu-driven CLI
- `catalog.json` – Book catalog storage file
